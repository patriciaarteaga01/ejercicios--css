Propiedades

Una propiedad en CSS es un atributo que especifica el estilo o el comportamiento de un elemento HTML, como el color, tamaño, márgenes o posición.

Modificar colores

color: define el color del texto.

background-color: define el color de fondo.

Modificar tamaños

width: define la anchura del elemento.

height: define la altura del elemento.

min-width: define la anchura miínima del elemento.

max-width: define la anchura máxima del elemento.

Texto

text-align: define el alineamiento del texto.

text-decoration: define la decoración del texto.

font-family: define la fuente del texto.

font-size: define el tamaño de la fuente.

font-weight: define el grueso de la fuente.
